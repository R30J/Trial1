What is C ?


C is a Programming Language. C is one of the oldest and first programming language. C was developed by Dennis Ritchie at At&T's Bell labs, USA in 1972.

1.Major parts of Windows, Linux and other OS are written in C.
2.It is used to write driver programs for devices like Tab, Printers.
3.C language is used to program embedded systems where programs need to
run faster in limited memory(Microwave, cameras).
4.C is used to develop Games an area where latency is very important.
Computer has to react quickly on user input.

    Input


    C supports a number of ways for taking user input. getchar() Returns the value of the next single character input.
    example:

    
            char a = getchar()
            printf("You entered: %c", a);

    gets()
    it is used to read input as an ordered sequence of characters, also called a string.
    example: 
    
            char a[100];
            gets(a);
            printf("You entered: %s", a);

    Here, we stored the input in an array of 100 characters..
    scanf()
    scans input that matches format specifiers.
    
            int a;
            scanf("%d", &a);
            printf("You entered: %d", a);

    & sign before the variable name is the address operator.it gives the address or location in memory. of a variable. This is needed because scanf places an input value at a variable address.


    *Formatted input


    scanf function is used to assign input to variables.

    
                int x;
                float num;
                char text[20];
                scanf("%d %f %s", &x, &num, text);

  • & must be used to access the variable adress isn't needed for a string because a string name acts as a pointer.
  • % used to assign values to corresponding arguments after the control string.Blanks, tabs, nuwlines are ignored.
  • %[*][max_field] conversion character * will skip the input field.
    max_width gives the max num of character to assign to an input field.
    • d decimal
    c character
    s string
    f float
    X hexadecimal

    example: 
    
            int x,y;
            char text[20];
            scanf("%2d %d %*f %55", &x, &y, text);
            /* input: 1234  5.7  elephant  */
            printf("%d %d %s", x, y, text);
            /* output: 12 34 eleph */

    Logical operator


    && Logical AND operator, if both the operands are non-zero then the condition is true.
    ex: (A&&B) is true.
    || Logical OR operator, if any of these two operands is non-zero, then condition becomes true.
    ex: (A||B) is true.
    ! Logical NOT operator, it is used to reverse the logical state of its operand. if condition is true, the logical NOT operator will make it false.
    !(A&&B) is ture.

    Category Operator Associativity
    Postfix () [] - > . + + - - Left to Right
    Unary + - ! ~ + + - - (type * & sizeof) Right to Left
    Multiplication * / % Left to Right
    Additive + - Left to Right
    shift < < > > Left to Right
    Relational < <= > >= Left fo Right
    Equality == != Left to Right
    Bitwise AND & Left to RIght
    Bitwise XOR ^ Left to Right
    Bitwise OR | Left or Right
    Logical AND && Left to Right
    Logical OR || Left to Right
    Conditional ?: Right to Left
    Assignment = += -= *= /= %=
    =>>= <<= &= ^= |=    		 Right to Left
